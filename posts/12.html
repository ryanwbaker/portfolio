Chem 209 Battlebox

November 19, 2019

This was a fun little project for CHEM 209, where we had to build an "escape room" style puzzle, that revealed the solution to a lock.

My Biggest Takeaways

  • This was my first time using an Arduino Nano and the Arduino programming software. I like it a lot! Very easy to use.
  • I figured out how to use software debouncing in order to make smooth button presses.
  • I think I'm going to look into digital displays, because 7-segment displays are a pain to wire up!
