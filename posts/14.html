My Biggest Takeaways

  • React is an incredibly efficient and organized way of building webpages compared to managing separate HTML, CSS, and JavaScript files.
  • The component-based architecture of React allows webpages to be broken down into small pieces that can be used over and over again.
  • Props are how data can be passed into React components. This is just the 'tip of the iceberg', but I'm very excited to continue learning React!
  • I found a great guide on how to publish a React app to GitHub Pages, which is what allowed me to quickly push this out to the web.
