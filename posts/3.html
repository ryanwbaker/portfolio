Looking Back on ENEL 327 (Signals and Transforms)

April 2, 2021
A Famous Quote By Donald Rumsfeld...

...goes like this (paraphrased): "There are things we know that we know. Then there are things we know that we don't know. Then there are things that we don't know that we don't know...". ENEL 327 was certainly in that final category. More than any other course that I can remember, this course covered a lot of concepts that I never really thought about before. Before studying signals and transforms, I've never really thought about why I was learning certain things in mathematics (like Laplace and Fourier transforms). ENEL 327 has definitely made those concepts clearer.

How I Would Explain This Course To A First Year Student

  • Everything you learn in math in the first three semesters leads to this class
  • Throughout the course, everything will feel very conceptual. By the end of the course, however, you will be able to see the practical applications of signal processing, and how important they are.
  • There are worksheets throughout the course that will help to touch on future applications of signal processing that we (as electrical engineering students) will encounter in the future

My Biggest Takeaway

There is a common core of mathematics in signal processing that has a wide diversity of applications. Anything from image processing, to communication transmission, to audio processing all involve the same mathemetical principles. This is pretty amazing if you think about it. I'm excited to delve into the applications of signal processing in future courses!

Applying Knowledge From This Course

Though this course was mostly conceptual, I can tell it lays the groundwork for many practical applications into the future, such as filter design (which was also taught in ENEL 343 [Circuits II]).
